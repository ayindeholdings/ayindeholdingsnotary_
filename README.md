# AyindeHoldings Notary Services

## Deploy on Vercel
1. Upload this folder to a GitHub repo or drag it into Vercel.
2. Vercel will detect Next.js automatically.
3. Click Deploy.

## Local run
```bash
npm install
npm run dev
```
